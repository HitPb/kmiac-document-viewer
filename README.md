# SOAPDocumentViewer
* получение документов по SOAP/WSDL
* поиск по полям
* просмотр документов

# Для работы программы требуется уставленный NodeJS

## Установка зависимостей

### Server

> cd api
> 
> npm i

### Front

> cd front
> 
> npm i

## Запуск

> npm run start

### Адрес фронта
> http://127.0.0.1:3000/

### Адрес сервера
> http://127.0.0.1:3334/
